Event Manager Web App
Overview

The Event Manager is a simple web application that allows users to create, view, and manage events dynamically. Users can add event details such as title, date, category, and description. Each event appears as a card on the screen and can be deleted individually or cleared completely.

The application demonstrates DOM manipulation, event handling, and dynamic element creation using JavaScript.

Features

Add new events with title, date, category, and description

Events displayed as dynamic cards

Delete individual events using the delete button

Clear all events with one click

Add sample event for testing

Responsive and clean UI design

Technologies Used

HTML5 – Structure of the webpage

CSS3 – Styling and layout

JavaScript (DOM Manipulation) – Dynamic functionality
